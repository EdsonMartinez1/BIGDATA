# Clasificación de Spam en Correos Electrónicos

## Objetivo

Desarrollar un modelo de aprendizaje supervisado capaz de clasificar mensajes como spam o legítimos utilizando técnicas de procesamiento de lenguaje natural.

## Dataset

SMS Spam Collection Dataset.

* Total de registros: 5572
* Ham: 4825
* Spam: 747

## Algoritmo utilizado

Naive Bayes

## Tecnologías

* Python
* Pandas
* Scikit-Learn
* Matplotlib
* Seaborn
* WordCloud

## Resultados

* Accuracy: 96.23%
* Precision: 100%
* Recall: 72%
* F1 Score: 83.72%
